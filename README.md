# NestMart – E-Commerce Homepage Wireframe

## Overview

NestMart is a low-fidelity e-commerce homepage wireframe designed to improve user experience through a clean and organized layout.

This project focuses on information hierarchy, user flow, and layout structure before applying colors, images, and visual styling.

The wireframe was created as part of a UI/UX Design Internship project using Figma.

---

## Problem Statement

Many e-commerce websites suffer from:

* Cluttered layouts
* Poor navigation
* Confusing user journeys
* Weak call-to-action placement
* Difficult product discovery

This project aims to create a simple and structured homepage that improves usability and user experience.

---

## Project Goal

The primary goal of this project is to design a low-fidelity homepage wireframe that focuses on:

* User flow
* Information hierarchy
* Content organization
* Clear navigation
* Effective CTA placement

## Design Approach

This project follows low-fidelity wireframing principles:

* Black, white, and grey color palette
* No real images
* Placeholder elements
* Focus on layout and structure
* Clear visual hierarchy
* Consistent spacing and alignment
